Sistem informasi rental mobil berbasis web menggunakan framework **CodeIgniter**. Aplikasi ini memudahkan pengelolaan data kendaraan, pemesanan, dan transaksi rental mobil.

---

## ✨ Fitur

- 🔐 Login & autentikasi pengguna
- 🚘 Manajemen data kendaraan (tambah, edit, hapus)
- 📋 Manajemen pemesanan rental
- 💰 Pengelolaan transaksi & pembayaran
- 📊 Laporan data rental
- 👥 Manajemen pengguna & hak akses

---

## 🛠️ Teknologi

- **Framework:** CodeIgniter (PHP)
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Library:** OwlCarousel, WOW.js, jQuery
- **Server:** Apache (Laragon)
